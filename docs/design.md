# 答案之书
每个问题都有答案

## 内容
### 生成答案之书
- 放在`public/`中
- 答案分4类：
  - 佛风
  - 儒风
  - 道风
  - 玄学风
- 每个答案每两部分
  - 一个符合对应风格的简短答案
  - 一个符合该答案及风格的解语（可以是广为人知的俗语、古典诗词、名言名句，佛家经典、儒学经典、道家经典、玄学经典，等等）
- 开放性答案
- 每个答案都有中英对照，先中文，再英文

### 编写代码
- webapp
- stateless server
- next.js + TS + Tailwind
- 所有内容都是中英对照，先中后英，本文的提示仅以中文写书，生成页面时，要将英文加进去

## 基本表现
- 主界面初始是一个大大的【开始】按钮，下边提示“心里默念问题，点击【开始】寻求答案“
  - 【开始】按钮是一个圆型结构，有特殊光效，在不同风格间切换
  - 提示内容风格亦会随之变化
- 点击【开始】之后
  - 【开始】按钮变成一个长方式按钮，内容变为【答案正在来临】，慢慢掉到底部
  - 与此同时，整个屏幕变成黑客帝国风格的竖式文本下降过程。有很多种答案慢慢下降，最终的答案慢慢变大变清晰，其它答案慢慢变淡并消失。文本从答案库中随机抽取。
  - 文本的总体下降速度比按钮的稍快。当按钮掉到底部时，文本已经足够铺满全屏。
  - 按钮掉到底部时，整个过程结束。按钮文本变为【再问一次】。最终答案由竖式变为横式，置于屏幕中间。
  - 整个过程大约2秒。
- 点【再问一次】，屏幕以时光倒流的形式，回退到初始状态。整个过程大约是1秒。
